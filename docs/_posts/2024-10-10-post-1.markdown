Creating this blog was not the most difficult process. But there were a few tricky things that I encountered right away. 

The first thing that I want to mention was that following the docs and trying to publish it on github-pages was an aboslute nightmare. 

I spent hours trying to configure the site through the docs and but I would always get an error upon viewing the site on github pages. 

It wasn't until I found the GitHub article on creating and publishing sites with Jekyll and Github Pages that I could figure it out. It then took less than an hour to get up and running. 

I'm not blaming the docs. I don't have the knowledge and experience to be critical about it. After all, I am a beginner and this is my first static site generator I'm using 

I was originally going to go with Hugo but then I learned about the strong integration that GitHub has with Jekyll
